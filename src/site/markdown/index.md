About
-----

Simple PSEi (formerly known as PHISIX) RESTful API hosted on Google AppEngine

**Resource URI**

    GET http://phisix-api.appspot.com/stocks.{json|xml}
    GET http://phisix-api.appspot.com/stocks/{symbol}.{json|xml}
